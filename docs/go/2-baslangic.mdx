# Go Kurulumu

Öncelikle [go.dev](https://go.dev) üzerinden gerekli dosyaları indiriyoruz.

:::info 

Ben bu kısımda Go'nun 1.19 sürümü ile çalışıyorum. Gerekirse kodlarda güncellemeye gidebilirim.

:::

Klasik kurulumdan sonra terminal üzerinden `go version` komutu ile kurulumu kontrol ediyoruz. Eğer sürüm geliyorsa sorun yok demektir. 

## VSCode Özelleştirme

Burada çok bir işlemimiz yok. Eklentilerden Go eklentisini kurmamız yeterli.
