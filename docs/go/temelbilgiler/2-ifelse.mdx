# If-else Yapısı

Go da diğer programlama dilleri gibi kendi içerisinde bir if-else yapısı bulundurur.

## if Komutu

Python gibi parantez istemeden kolay bir şekilde tanımlama yapabiliriz. 

```go
package main

import "fmt"

func main() {
	var balance = 100
    var price = 50

    if balance > price{
        fmt.Println("You have enough money to buy the product")
    }
}
```
